# 恐竜クイズ

恐竜に関する知識を楽しく学べるPWAアプリケーション

## 機能

- 恐竜のランキング表示
- 恐竜の画像表示
- PWA対応（オフライン動作可能）
- レスポンシブデザイン

## 技術スタック

- Next.js
- TypeScript
- Chakra UI
- PWA (next-pwa)

## 開発環境のセットアップ

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# 本番サーバーの起動
npm run start
```

## デプロイ

このプロジェクトはVercelにデプロイすることを想定しています。
以下の手順でデプロイできます：

1. Vercelアカウントを作成
2. Vercel CLIをインストール: `npm i -g vercel`
3. デプロイ: `vercel`
